40 Datos con 10 Keys recurrentes

10 comandos

Comando 1: db.materias.find({ nombre: { $eq: "programacion orientada a objetos" } });

muestra las materias con el nombre "programacion orientada a objetos

Comando 2: db.materias.find({ creditos: { $gt: 2 } });

muestra las materias que tienen mas de 3 creditos

Comando 3: db.materias.find({ horas_clases_teoricas: { $gte: 3 } });

muestra las materias que tienen 3 horas o mas de clases teroicas

Comando 4: db.materias.find({ horas_clases_practicas: { $lt: 1 } });

muestra las materias que tienen menos de 1 hora de clases prácticas

Comando 5: db.materias.find({ codigo: { $lte: '105' } })

muestra las materias en donde su codigo es menor o igual a '105' tomando un valor numerico

Comando 6: db.materias.find({ profesores_responsables: { $ne: 'luis castro' } });

muestra las materias que no tienen como profesor a 'luis castro'

Comando 7: db.materias.find({ aulas_asignadas: { $in: ['aula 2101', 'aula 2201', 'aula 3301', 'aula 3001', 'aula 2801', 'aula 2301', 'aula 3901' ] } });

muestra las materias que se encuentran en las aulas 'aula 2101', 'aula 2201', 'aula 3301', 'aula 3001', 'aula 2801', 'aula 2301', 'aula 3901'

Comando 8: db.materias.find({ promedio_calificaciones: { $nin: [4.5] } });

muestra las materias que no tienen promedio de 4.5

Comando 9: db.materias.find();

muestra todas las materias de la base de datos

Comando 10: db.materias.deleteOne({ nombre: "filosofia del derecho" });

elimina la materia llamada "filosofia del derecho"
