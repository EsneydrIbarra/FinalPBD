40 Datos con 10 Keys recurrentes

10 comandos

Comando 1: db.hoteles.find({ nombre: { $eq: "hotel el lago" } })

muestra un hotel con el nombre "hotel el lago"

Comando 2: db.hoteles.find({ clasificacion: { $gt: "4 estrellas" } })

muestra hoteles que su clasificación sea mayor que 4 estrellas

Comando 3: db.hoteles.find({ tarifa_promedio: { $gte: 500000 } })

muestra hoteles donde su tarifa promedio sea mayor o igual a 500.000

Comando 4: db.hoteles.find({ area_total: { $lt: 1000 } })

muestra hoteles en donde su área total sea menor que 1000 metros cuadrados

Comando 5: db.hoteles.find({ capacidad_habitacion: { $lte: 2 } })

muestra hoteles con capacidad de personas menor o igual a 2 personas por cuarto

Comando 6: db.hoteles.find({ ubicacion: { $not: /pasto/i } })

muestra hoteles que no están ubicados en la ciudad de Pasto con el operador logico, not, con $ne no funciono

Comando 7: db.hoteles.find({ ingresos_mensuales: { $in: [15000000, 20000000, 25000000, 30000000] } })

muestra hoteles cuyos ingresos mensuales estén entre 15,000,000 y 30,000,000

Comando 8: db.hoteles.find({ tarifa_promedio: { $nin: [50000, 60000, 70000, 80000, 90000, 100000, 110000, 120000, 150000, 180000, 200000, 250000, 300000, 350000] } })

muestra hoteles cuya tarifa promedio no esté en los valores dados

Comando 9: db.hoteles.find()

este comando muestra todos los hoteles en la base de datos

Comando 10: db.hoteles.deleteOne({ nombre: "hotel colonial" })

elimina el hotel llamado "hotel colonial"
